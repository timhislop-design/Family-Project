# Drafts — Frozen Snapshots

We're experimenting, so we never throw a version away. The **live files** (one level up: `story.md`, `outline.md`, `characters/`, etc.) are always the current working version. This folder holds **frozen snapshots** — restore points we can always go back to.

## How it works
- Each snapshot is a dated, numbered folder: `YYYY-MM-DD_vN/`.
- A snapshot is a *copy* of the live files at that moment. We never edit snapshots.
- To go back, say **"restore [folder name]"** and Claude brings that version back to the live files.

## Saving a snapshot
Just say **"save a draft"** (optionally "...call it [label]") and Claude freezes the current state into a new dated folder here.
